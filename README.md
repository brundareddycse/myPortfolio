# Brunda Reddy - Portfolio

A modern, responsive portfolio website built with React, Tailwind CSS, and Express.

## Features
- **Hero Section**: Strong personal branding with bio and tagline.
- **Projects**: Showcase of key projects with details and links.
- **Skills**: Categorized technical skills.
- **Experience**: Timeline of education and professional experience.
- **Contact**: Functional contact form.
- **Responsive Design**: Works perfectly on mobile and desktop.
- **Animations**: Smooth transitions using Framer Motion.

## Tech Stack
- **Frontend**: React, Tailwind CSS, Framer Motion, Lucide React
- **Backend**: Node.js, Express
- **Database**: PostgreSQL (Drizzle ORM)

## Setup
1. Install dependencies: `npm install`
2. Push database schema: `npm run db:push`
3. Start the server: `npm run dev`

## Assets
- **Icons**: [Lucide React](https://lucide.dev)
- **Fonts**: Inter, Space Grotesk
- **Colors**:
  - Primary: `hsl(222, 47%, 11%)`
  - Accent: `hsl(262, 80%, 50%)`

## Deployment
Ready to deploy on Replit, Vercel, or Netlify.
